# Intelligent Systems Practice

山东大学计算机科学与技术学院 24 智能班智能系统综合实践实验

> 事实上，此课程为早几年数据结构的课设部分，被单独拿了出来
>
> 此课设原本为 part1:固定 4 道小题目 + part2:题库随机抽取 1 道大题目组成
>
> 但在 24 级，改为了固定的 part1:1 道小题目 + part2:2 选 1 道大题目
>
> PS:我认为可能是前几届学长发力了

## Warning

题目没有给出样例或者是任何的数据规模相关的提示,因此所有数据类型及规模都是假设的
